### 3.2 Resource Fields
| Field | Description |
|-------|-------------|
| Course ID | Links to curriculum |
| Title | Resource title |
| Type | Video, audio, document |
| Format | MP4, PDF, etc. |
| URL | Storage location |
| Duration | For video/audio |
| Page Count | For documents |

### 4.1 Lecturer Features
- Upload videos (MP4, WebM)
- Upload documents (PDF, PPTX)
- Upload audio (MP3)
- Create assignments
- Set due dates
- Manage resources

| Images | JPG, PNG, GIF |
