# LEARNING ANALYTICS REPORT

**InnovaSci AI Labs Polytechnic**
**Learning Resource Access Engine**

---

## 1. Analytics Overview

### 1.1 Purpose
Track and analyze student learning activities to improve educational outcomes and ensure ODFeL compliance.

### 1.2 Tracked Metrics
| Metric | Description |
|--------|-------------|
| Learning Time | Total time spent on learning |
| Courses Accessed | Number of courses accessed |
| Videos Watched | Video viewing statistics |
| Documents Downloaded | Document download count |
| Notes Created | Personal notes created |
| Assignments Completed | Assignment submissions |

---

## 2. Student Analytics

### 2.1 Individual Tracking
Each student's learning activities are tracked:
- Login timestamps
- Course access times
- Video watch duration
- Document downloads
- Notes created
- Assignment submissions

### 2.2 Progress Metrics
| Metric | Calculation |
|--------|-------------|
| Course Progress | Courses accessed / Total courses |
| Video Completion | Videos watched / Total videos |
| Assignment Rate | Assignments submitted / Total assignments |
| Average Progress | Overall learning progress % |

---

## 3. Programme Analytics

### 3.1 Programme-Level Statistics
- Average learning time per programme
- Resource utilization rates
- Popular courses
- Engagement metrics

### 3.2 Faculty Analytics
Aggregate statistics for each faculty:
- Total learning hours
- Resource consumption
- Completion rates

---

## 4. ODFeL Compliance Metrics

### 4.1 Distance Learning Tracking
- Remote access statistics
- Learning time distribution
- Participation rates
- Progress tracking

### 4.2 Compliance Indicators
| Indicator | Target |
|-----------|--------|
| Active Learners | >80% |
| Course Completion | >70% |
| Assignment Submission | >85% |
| Video Engagement | >60% |
