## 1. Practical Learning Model

Every laboratory activity follows a structured practical learning workflow:

```
┌─────────────────────────────────────────────────────────────┐
│                    LEARN                                    │
│  - Read lab manual                                          │
│  - Watch demonstration videos                               │
│  - Review procedure guides                                  │
│  - Understand safety instructions                          │
└─────────────────────────────────────────────────────────────┘
                           ↓
┌─────────────────────────────────────────────────────────────┐
│              PERFORM PRACTICAL ACTIVITY                     │
│  - Execute experiment steps                                │
│  - Record measurements                                     │
│  - Observe results                                        │
│  - Follow safety protocols                                 │
└─────────────────────────────────────────────────────────────┘
                           ↓
┌─────────────────────────────────────────────────────────────┐
│               RECORD OBSERVATIONS                           │
│  - Digital Lab Notebook entries                            │
│  - Capture calculations                                     │
│  - Save findings                                           │
│  - Document media (photos, data)                           │
└─────────────────────────────────────────────────────────────┘
                           ↓
┌─────────────────────────────────────────────────────────────┐
│                 ANALYZE RESULTS                             │
│  - Process experimental data                               │
│  - Create charts and visualizations                       │
│  - Identify patterns                                       │
│  - Draw conclusions                                        │
└─────────────────────────────────────────────────────────────┘
                           ↓
┌─────────────────────────────────────────────────────────────┐
│              GENERATE LAB REPORT                             │
│  - Compile observations                                     │
│  - Document procedures                                      │
│  - Present results                                          │
│  - Write analysis and conclusions                          │
└─────────────────────────────────────────────────────────────┘
                           ↓
┌─────────────────────────────────────────────────────────────┐
│                SUBMIT LAB REPORT                             │
│  - Review for completeness                                  │
│  - Submit for review                                        │
│  - Await lecturer feedback                                  │
└─────────────────────────────────────────────────────────────┘
                           ↓
┌─────────────────────────────────────────────────────────────┐
│              PRACTICAL ASSESSMENT                            │
│  - Take practical quiz                                      │
│  - Answer viva questions                                    │
│  - Complete competency assessment                           │
│  - Receive score                                           │
└─────────────────────────────────────────────────────────────┘
```

## 2. Lab Report Generation

### 2.1 Report Sections

| Section | Content Source |
|---------|----------------|
| Student Information | Auto from student profile |
| Course Information | Auto from curriculum mapping |
| Experiment Title | Activity title |
| Date | Auto timestamp |
| Objectives | Activity objectives |
| Materials | Activity materials list |
| Procedures | Step-by-step procedures |
| Observations | Student notebook entries |
| Results | Experimental data |
| Analysis | Student analysis |
| Conclusion | Student summary |

### 2.2 Report Status Flow

```
DRAFT → SUBMITTED → REVIEWED → APPROVED
                    ↓
                RESUBMIT (if rejected)
```

## 3. Digital Lab Notebook

### 3.1 Notebook Entry Types

| Type | Purpose |
|------|---------|
| Observation | Record experimental observations |
| Calculation | Document mathematical work |
| Finding | Record key discoveries |
| Note | General study notes |
| Sketch | Diagrams and drawings |

### 3.2 Notebook Integration

Each notebook entry is linked to:
- Course code
- Laboratory activity
- Student ID
- Timestamp

---

## 4. Assessment Integration

### 4.1 Assessment Types

| Type | Description |
|------|-------------|
| Practical Quiz | Multiple choice, short answer |
| Viva Questions | Oral examination |
| Laboratory Assignment | Extended practical tasks |
| Laboratory Report | Written documentation |
| Competency Assessment | Skill evaluation |

### 4.2 Competency Levels

| Level | Criteria |
|-------|----------|
| Beginner | < 3 activities completed |
| Intermediate | 3-5 activities, avg score < 65% |
| Proficient | 6+ activities, avg score >= 65% |
| Advanced | 6+ activities, avg score >= 80% |
